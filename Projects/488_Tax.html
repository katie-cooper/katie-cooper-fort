SmartTax

AI-powered chatbot for tax assistance and filing.

Spring 2025

Overview & Functionality

SmartTax is an AI-powered chatbot designed to assist users with tax-related questions and simplify tax information for young adults and first-time filers. Built with authoritative data sources and enhanced with a supportive interaction model, SmartTax ensures a seamless and less stressful tax filing experience.

Technical Skills & Tech Stack

SmartTax was developed as part of an advanced AI applications course project, requiring the design of a complete AI-driven service with real-world utility. Building this product involved integrating generative AI, retrieval-augmented generation (RAG), and a web interface backed by custom logic and APIs.

Objectives

  • Help young adults and first-time tax filers understand basic tax concepts.
  • Provide clear, concise, and up-to-date tax information.
  • Offer guidance on tax forms, filing requirements, and available deductions.
  • Use accessible language to make tax information easier to understand.
  • Be a supportive and encouraging tool for users navigating the complexities of tax filing.

Data Sources & Curation

1. IRS Documentation

  • IRS forms and handouts detailing filing requirements, deductions, and credits.
  • Official IRS publications ensuring accuracy and compliance with tax laws.

2. Trusted Tax Assistance Companies

  • TurboTax and NerdWallet provide simplified tax explanations in an accessible tone.
  • These sources help ensure the chatbot delivers tax information in a way that is easier to understand compared to formal IRS documentation.

Features & Functionalities

  • Intelligent Response Tone: Provides clear and well-structured answers to tax-related inquiries.
  • Sentiment Analysis & Hostility Mitigation: Detects hostile or frustrated user inputs and responds appropriately.
  • Encouraging & Supportive Interactions: Reassures users that they are capable of successfully filing their taxes.

Core Technical Skills Applied

  • AI/ML Integration: Designed and fine-tuned an AI assistant capable of handling tax-related queries using GenAI models and RAG pipelines.
  • Natural Language Processing: Implemented structured responses with sentiment detection and tone adjustment for supportive interactions.
  • Backend Development: Built backend logic to handle query processing, data source retrieval, and response orchestration.
  • API Integration: Connected IRS datasets and trusted third-party tax resources into the chatbot’s knowledge base.
  • Web Development: Created a user-friendly frontend interface for real-time interactions.
  • Version Control & Collaboration: Used GitHub for source control and project management.

Tech Stack

  • Frontend: HTML, CSS, and JavaScript for the web interface.
  • Backend: Python (FastAPI) for API endpoints and orchestration logic.
  • AI Models: Generative AI with Retrieval-Augmented Generation (RAG) for accurate, source-grounded answers.
  • Database / Knowledge Sources: IRS publications, tax handouts, TurboTax & NerdWallet for simplified tax guidance.
  • Deployment: Vercel for hosting the web interface; GitHub for version control.

Through this project I gained hands-on experience combining AI model integration, data curation, and full-stack development to deliver a usable product — demonstrating how emerging AI technologies can be applied to build accessible and practical tools.

Future Enhancements

  • Expanded Knowledge Base: Inclusion of more state-specific tax regulations and international tax guidance.
  • Integration with Tax Filing Software: Direct links to tax preparation tools for a seamless experience.
  • Personalized Assistance: Machine learning improvements to provide personalized tax advice based on user history and preferences.
